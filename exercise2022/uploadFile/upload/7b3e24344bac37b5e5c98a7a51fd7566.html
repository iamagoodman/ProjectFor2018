基本信息

Path： /obtainSmsVerificationCode

Method： POST

接口描述：

接口描述：
测试环境url: xzddns.f3322.net:8080/open-art/api/obtainSmsVerificationCode

请求示例

{
    "sid":"sLskCxBLDhRcYzZpXCtpaVnrGrLsUftHQjPxWMLfHUaddBrNhLhONccLHPhsKDNbhiCDftFXOcQRYVtyiSVJdAydVloDUBNDeKOgaIrEUpNqRaRKzHqEhRPZBwXicjak",
    "inputImgCode":"k70i",
    "phoneNumber":"18236592118"
}

返回示例

{
  "code": 200,
  "msg": "success",
  "data": {
    "smsSerialNumber": "995539",
    "smsCode": "070217"
  }
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是

Body

名称类型是否必须默认值备注其他信息
sidstring必须安全码
inputImgCodestring必须图形验证码
phoneNumberstring必须手机号

返回数据

名称类型是否必须默认值备注其他信息
codestring必须
msgstring必须
dataobject必须
├─ smsSerialNumberstring必须短信编码
├─ smsCodestring必须短信验证码。后期通过短信发送
